# 🎨 초경량 이미지 to ICO 변환기 (Image to ICO Converter)

복잡한 설치나 무거운 프로그램 없이, 단 하나의 HTML 파일로 이미지를 고품질 아이콘(.ico) 파일로 변환하는 도구입니다.

## 📦 프로젝트 정보

- **파일 위치**: `index.html`
- **용량**: 약 **15KB** (단일 파일)
- **개발 언어**: HTML5, CSS3, Vanilla JavaScript

## 🎯 주요 기능

- ✅ **드래그 앤 드롭**: 이미지를 브라우저 창에 끌어다 놓기만 하면 인식됩니다.
- ✅ **파일 선택 버튼**: 클릭하여 폴더에서 파일을 찾을 수도 있습니다.
- ✅ **실시간 미리보기**: 변환 전 이미지를 미리 확인할 수 있습니다.
- ✅ **멀티 사이즈 지원**: 16x16부터 256x256까지 다양한 크기를 선택하여 하나의 ICO 파일에 담을 수 있습니다.
- ✅ **완전 오프라인**: 인터넷 연결이 필요 없으며, 데이터가 서버로 전송되지 않아 **보안상 안전**합니다.
- ✅ **프리미엄 디자인**: 눈이 편안한 다크 모드와 부드러운 애니메이션이 적용된 모던 UI를 제공합니다.

## 🚀 사용 방법

1. `index.html` 파일을 더블클릭하여 브라우저(Chrome, Edge 등)에서 엽니다.
2. 변환하고 싶은 이미지(JPG, PNG)를 중앙 영역에 드롭하거나 클릭하여 선택합니다.
3. 원하는 아이콘 크기(16, 32, 48, 64 등)를 체크박스에서 선택합니다.
4. **"💾 ICO로 변환"** 버튼을 클릭합니다.
5. 변환된 `icon.ico` 파일이 즉시 다운로드됩니다.

## 🎨 특징

- **초경량**: 불필요한 라이브러리 없이 순수 웹 기술로 제작되어 매우 가볍습니다.
- **반응형 디자인**: PC는 물론 모바일 환경에서도 최적화된 화면을 보여줍니다.
- **설치 불필요**: `exe` 파일처럼 설치 과정을 거칠 필요가 없습니다. 파일 하나만 있으면 어디서든 실행 가능합니다.

## 🛠️ 작동 원리

이 프로그램은 별도의 백엔드 서버 없이 브라우저의 자체 기능만을 활용합니다:
1. **Canvas API**: 이미지를 불러와 브라우저 내부에서 다양한 크기로 리사이징합니다.
2. **Binary Manipulation**: 리사이징된 이미지 데이터를 ICO 포맷 규격에 맞춰 바이너리(0과 1) 데이터로 재조립합니다.
3. **Blob & Download**: 메모리 상에서 생성된 파일을 브라우저의 다운로드 기능을 통해 PC에 저장합니다.
